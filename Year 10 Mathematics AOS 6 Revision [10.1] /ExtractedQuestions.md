### Question 6 (2 marks)

**Question Text:**
Consider the Venn diagram below:

![Venn Diagram](images/diagram_page04_q6_venn.png)

**Diagram Description:**
- Type: Two-circle Venn diagram
- Set A (left circle): 12 in A only, 5 in intersection

- Outside both circles: 8 (neither)
- Total universe: 40 (12 + 5 + 15 + 8)
- Style: Simple overlapping circles with counts in each region

**Critical Diagram Elements:**

| Element | Status | Notes |
|---------|--------|-------|
| **MUST KEEP** | Two overlapping circles | Core structure for testing union concept |
| **MUST KEEP** | Four distinct regions (A only, B only, Both, Neither) | Essential for P(A ∪ B) calculation |
| **MUST KEEP** | Clear numerical values in each region | Students must read and add values |
| **CAN CHANGE** | Specific numbers in each region | Can vary: 12→17, 5→8, 15→12, 8→13 (as long as total adds up) |

**Testing Aspect:**

| Aspect | Details |
|--------|---------|
| **Concept** | Union of sets using Venn diagram |
| **Formula** | P(A ∪ B) = (A only + Both + B only) / Total OR P(A) + P(B) - P(A ∩ B) |
| **Skill Being Tested** | Read Venn diagram regions, calculate union probability |
| **Understanding Required** | Union means "A or B or both", must add all regions except "neither" |

### Question 7 (2 marks)

**Question Text:**
If P(A) = 0.6, P(B) = 0.5 and P(A ∪ B) = 0.8, find P(A ∩ B).

**Testing Aspect:**

| Aspect | Details |
|--------|---------|
| **Concept** | Addition rule for probability (working backwards) |
| **Formula** | P(A ∪ B) = P(A) + P(B) - P(A ∩ B), rearranged to find P(A ∩ B) |
| **Skill Being Tested** | Rearrange addition rule formula, substitute values |
| **Understanding Required** | Relationship between union, individual probabilities, and intersection |

**For Question Generator:**
- Intermediate difficulty (2 marks, algebraic rearrangement)
- Give three probabilities, find the fourth
- Use decimals (0.0-1.0) or fractions
- Ensure values are consistent (P(A ∪ B) ≤ P(A) + P(B))
- Priority 1+2: Give different three probabilities and ask for the fourth using addition rule

**For Question Generator:**

- Use fractions or decimals
- Answer requires: calculation + statement (independent/dependent)
- Priority 1+2: Give different probabilities and test independence

### Question 13 (3 marks) Tech-Active

| Aspect | Details |

| **Understanding Required** | Variance measures spread, SD is in original units |

**For Question Generator:**

- Mean typically given or easy to calculate
- Part (a): Variance in exact form (fraction or integer)
- Part (b): SD to 2 decimal places
- Priority 1+2: Change dataset, potentially ask for sample SD instead (using n-1 divisor from curriculum)
